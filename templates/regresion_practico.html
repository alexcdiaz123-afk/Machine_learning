Ejercicio Práctico: Predicción de Peso

Explicación del dataset de entrenamiento

El dataset contiene 200 registros simulados con tres variables principales:

  • Estatura (m): altura en metros de las personas.
  • Edad (años): edad de los individuos (entre 5 y 90 años).
  • Peso (kg): calculado a partir de una fórmula lineal con ruido aleatorio.

Este dataset se generó artificialmente para entrenar un modelo de Regresión Lineal Múltiple que predice el peso a partir de la estatura y la edad.
